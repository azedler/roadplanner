# License decision before public HACS distribution

HACS cannot install from a private GitHub repository. Before switching the repository to public, choose an explicit license.

Common options:

- **MIT:** simple, permissive, allows commercial reuse and forks.
- **Apache-2.0:** permissive with an explicit patent grant.
- **GPL-3.0:** derivatives distributed to others must remain under GPL.
- **Source-available/custom commercial license:** requires legal drafting and is not an open-source license.

Until the decision is made, the repository should remain private and the root `LICENSE` file states that all rights are reserved.
