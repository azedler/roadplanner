# License policy

Roadplanner is licensed under the Apache License, Version 2.0.

## Project files

- The full license text is stored in the repository root as `LICENSE`.
- Project attribution is stored in `NOTICE`.
- Release artifacts must include both files.
- Contributions are accepted under Apache-2.0 unless explicitly marked otherwise before submission.

## Third-party material

Do not add code, media, documentation, fonts, datasets, screenshots, generated assets, or dependencies with unclear provenance.

For every third-party dependency or bundled asset, verify:

- its license permits the intended use and redistribution,
- required copyright and attribution notices are retained,
- NOTICE obligations are documented,
- source-offer or copyleft obligations are understood before inclusion.

## AI-generated contributions

AI-generated output is untrusted input. Review it for copied material, incompatible dependencies, license obligations, private data, secrets, and unexpected provenance before committing it.
