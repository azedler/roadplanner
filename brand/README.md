# Roadplanner brand assets

This directory contains the repository-level assets used by HACS.

- `icon.png` is the square Roadplanner icon required for HACS repository presentation.
- `logo.png` is the horizontal Roadplanner logo.

Runtime copies remain under `custom_components/roadplanner_mcp/brand/` because the Home Assistant panel also uses them. Keep both locations synchronized when the brand changes.
